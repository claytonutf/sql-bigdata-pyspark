# SQL - Big Data - PySpark
### Professor/Mentor: Clayton Kossoski
### UTFPR e Hub de IA do Senai — 2025

Este repositório contém exemplos e notebooks para trabalhar com **SQL** usando **Pandas DataFrame** e **Big Data** usando **PySpark**.

## Conteúdo do Repositório

- `pyspark-sql.ipynb` - Notebook em PySpark mostrando operações SQL em DataFrames.
- `python-sql.ipynb` - Notebook em Python com consultas SQL em datasets.
- `employees.csv` - Dataset de exemplo utilizado nos notebooks.
- `apresentacoes/` - Pastas com apresentações relacionadas ao projeto.
- `docker-compose.yml` - Arquivo para configuração de ambientes Docker.
- `LICENSE` - Licença do repositório (CC-BY-4.0).

## Pré-requisitos

1. Python 3.8+  
2. PySpark  
3. Jupyter Notebook  
4. Docker (opcional, se for usar o `docker-compose.yml`)  
34. Conhecimento básico em SQL (opcional, mas recomendado para melhor entendimento dos notebooks)
